Inlugg's Input Lag Investigation

This project is a technical deep-dive into unexplained input lag. My research focuses on how electrical interference (EMI), lack of grounding, and high-end PSUs (like MSI MPG A750GF) affect modern gaming peripherals.

Current Observations

I've noticed a "building up" effect. After some time, the mouse sensor responsiveness degrades. This correlates with static shocks from the metal microphone and case.

Hardware Performance Comparison

Pulsar X2

Status: Inconsistent

Sensitive controller reacts poorly to electrical noise. Result: 2 - 10 in Valorant.

Razer DeathAdder V2

Status: Stable

More robust EMI shielding. Result: 19 - 14 in the same environment.

Electrical Setup

  • PSU: MSI MPG A750GF (Gold, high-frequency filtering).
  • Condition: No grounding (Ground wire missing).
  • Observation: Static electricity builds up on the USB GND bus.
